# AI Infrastructure Cost Simulator

An interactive web application that compares token-based AI pricing models with compute-based infrastructure costs (NCU/GPU models).

## Purpose

Helps founders, operators, and decision-makers understand when token pricing becomes expensive versus owning or leasing compute resources.

## Live Demo

https://compute-vs-token-demo.vercel.app

## Key Features

- Compare token costs vs compute costs
- Dynamic pricing scenarios
- Usage-based simulation
- Business decision support visuals
- Simple interactive UI

## Use Cases

- AI startups evaluating hosting costs
- SaaS founders planning margins
- Product teams estimating inference spend
- Investors analyzing AI unit economics

## Tech Stack

- Python
- HTML/CSS
- JavaScript
- Vercel Deployment

## Why This Matters

Many companies underestimate long-term token pricing costs. This tool helps compare build-vs-buy economics in practical terms.
